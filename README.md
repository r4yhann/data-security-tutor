# DataSec Academy (data-security-tutor) 🛡️
> Plataforma interactiva de capacitación en ciberseguridad y gestión de datos.

Este proyecto es una plataforma educativa diseñada para enseñar conceptos críticos de seguridad de la información, enfocándose en el ciclo de vida del dato y la protección de la privacidad.

## 🚀 Características
- **Módulos Interactivos:** Tutoriales paso a paso sobre borrado seguro y normativas.
- **Simulador de Terminal:** Prácticas de comandos para gestión de políticas y auditoría.
- **Evaluación Final:** Sistema de examen para validar el conocimiento adquirido.
- **Diseño Responsive:** Optimizado para dispositivos móviles y escritorio.

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 (Custom Theme) y Bootstrap 5.
- **Lógica:** JavaScript ES6+ (Sin dependencias externas pesadas).
- **Despliegue:** Cloudflare Pages & Workers para alta disponibilidad.
- **Monetización:** Integración con Google AdSense mediante Ads.txt.

## 📋 Estructura del Proyecto
- `3.1` Seguridad en Gestión de Datos (RBAC y Auditoría).
- `3.2` Técnicas de Borrado (Lógico vs. Físico).
- `3.3` Prevención de Filtraciones (DLP).
- `3.4` Privacidad de Datos y Marcos Legales.
