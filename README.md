# Data Security Interactive Tutor

Este proyecto es una plataforma interactiva diseñada para enseñar conceptos críticos de ciberseguridad y gestión de datos. 

## 🚀 Características
- Tutoriales paso a paso sobre borrado seguro.
- Guías interactivas de políticas de acceso (RBAC).
- Simuladores de remanencia de datos.

## 🛠️ Tecnologías
- **Frontend:** HTML5, Bootstrap 5.
- **Interactividad:** JavaScript.
- **Despliegue:** GitHub Pages + Cloudflared.

## 📋 Contenidos
1. Seguridad en Gestión de Datos.
2. Técnicas de Borrado y Destrucción.
3. Prevención de Filtraciones (DLP).
4. Privacidad de Datos.
